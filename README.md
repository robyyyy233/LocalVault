# LocalVault (Password Manager)

**Status:** Early development / In progress  
**Type:** Learning project  

LocalVault is an offline, local-only password manager built as a learning
project to explore vault file design, encryption workflows, and desktop UX.

## ⚠️ Security Notice
This project is **not complete** and **not secure** in its current state.
Encryption and key derivation are still under development.

Do **not** use this project to store real passwords or sensitive data.
